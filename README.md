# منصة الأمن السيبراني الدفاعي 🛡️
# Cybersecurity Defensive Scanning Platform

## 📋 نظرة عامة
منصة متكاملة متخصصة في فحص الأمن السيبراني الدفاعي، تقوم بفحص المواقع والملفات والروابط المشبوهة والكشف عن التهديدات الأمنية.

## ✨ المميزات الرئيسية
- ✅ فحص المواقع الإلكترونية (URL Scanning)
- ✅ فحص الملفات والبرامج (File Malware Detection)
- ✅ تحليل الروابط المشبوهة (Link Analysis)
- ✅ قاعدة بيانات متقدمة للتهديدات
- ✅ لوحة تحكم حديثة
- ✅ تقارير مفصلة وآنية
- ✅ API متكاملة
- ✅ نظام إنذارات

## 🏗️ هيكل المشروع
```
Ii/
├── backend/              # الواجهة الخلفية (Flask)
├── frontend/             # الواجهة الأمامية (React)
├── database/             # قاعدة البيانات
├── api/                  # API المشروع
├── config/               # ملفات التكوين
├── requirements.txt      # مكتبات Python
├── package.json          # مكتبات Node.js
└── README.md            # التوثيق
```

## 🚀 البدء السريع

### المتطلبات
- Python 3.8+
- Node.js 14+
- PostgreSQL

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/AlexiONR/Ii.git
cd Ii

# تثبيت المكتبات
pip install -r requirements.txt
npm install

# تشغيل الخادم
python backend/app.py

# تشغيل الواجهة الأمامية
npm start
```
